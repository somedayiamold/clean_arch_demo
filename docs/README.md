# docs/

全局文档。

## 目录结构

```
docs/
├── adr/              架构决策记录（Architecture Decision Records）
├── conventions/      编码规范 & 提交规范
└── onboarding.md     新人入门指南
```

## ADR 模板

每条 ADR 使用如下格式：

```markdown
# ADR-{number}: {title}

## 状态
Accepted / Deprecated / Superseded

## 上下文
{描述问题背景}

## 决策
{描述所做的决策}

## 后果
{描述决策带来的正面和负面影响}
```
